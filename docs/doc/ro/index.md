# Ce este QualCoder

QualCoder este un program gratuit, open source, destinat analizării datelor calitative.

QualCoder permite codificarea textului, imaginilor, fișierelor audio și video, precum și redactarea de note de jurnal și memorii. Puteți organiza codurile într-o schemă de categorizare de tip arbore. Se pot genera diverse rapoarte. Majoritatea rapoartelor pot fi exportate în format HTML, OpenDocument Text (ODT), text simplu sau fișiere Excel.

QualCoder include un set de funcții asistate de IA care utilizează modele lingvistice de mari dimensiuni pentru a vă ajuta să explorați, să analizați și să interacționați cu datele în moduri inovatoare. Familiarizați-vă cu datele noi explorând subiecte sau concepte generale într-o conversație interactivă cu IA. Aprofundați aspecte specifice cu ajutorul codificării și analizei textului asistate de IA. Iar dacă ajungeți în punctul în care trebuie să sintetizați și să consolidați rezultatele, discutați-le într-o conversație despre coduri cu IA.

!!! note "De ce să folosiți QualCoder?"
    - **QualCoder este gratuit**. Multe programe de analiză calitativă necesită taxe unice costisitoare sau abonamente lunare.
    - **QualCoder este ușor de utilizat**. Are tot ce aveți nevoie pentru a efectua analize calitative fără interfețele complicate ale unor alternative
    - **QualCoder funcționează offline**. Internetul nu este întotdeauna disponibil, iar QualCoder nu necesită conexiune la internet pentru a funcționa.
    - **QualCoder nu este legat de un anumit computer**. Dacă vă schimbați locul de muncă, nu trebuie să vă faceți griji că veți rămâne legați de licența fostului loc de muncă sau că va trebui să cumpărați o nouă licență. Licența QualCoder vă permite să utilizați software-ul indiferent de locul în care lucrați sau de computerul pe care este instalat.
    - **QualCoder este multiplataformă**. Funcționează pe Linux, Windows și macOS, ceea ce înseamnă că nu trebuie să vă faceți griji dacă schimbați sistemul de operare și, de asemenea, că puteți colabora cu colegii care utilizează platforme diferite. A se vedea Nota.
    - **QualCoder se bazează pe comunitate**. Dacă descoperiți o eroare, aveți o solicitare de funcționalitate sau doriți să oferiți feedback, scrieți despre aceasta pe [pagina QualCoder de pe GitHub](https://github.com/ccbogel/QualCoder).
    - **QualCoder se îmbunătățește continuu**. QualCoder este dezvoltat în mod activ, ceea ce înseamnă că sunt lansate versiuni noi și îmbunătățite.
    - **QualCoder acceptă standarde deschise**. QualCoder își propune să accepte [standardul REFI-QDA](https://www.qdasoftware.org/); puteți face schimb de coduri și proiecte cu colegii dvs. chiar dacă aceștia nu utilizează QualCoder, atâta timp cât software-ul pe care îl folosesc acceptă standardul REFI-QDA. Aceasta înseamnă că nu riscați ca datele dvs. să devină indisponibile. Proiectul REFI-QDA nu este încă garantat ca fiind 100% conform. Sunt necesare teste suplimentare, în special pentru exporturile și importurile de fișiere audio și video, precum și pentru fișierele legate relativ.
    - **QualCoder poate fi modificat**. Puteți modifica și adapta QualCoder la nevoile dumneavoastră, atâta timp cât puneți la dispoziția tuturor modificările efectuate. Acest lucru înseamnă, de asemenea, că puteți copia software-ul și îl puteți oferi colegilor sau studenților dumneavoastră în mod gratuit.
    - **QualCoder este o opțiune software care susține știința deschisă**. [Recomandările UNESCO privind știința deschisă](https://www.unesco.org/en/open-science?hub=686)
