# Che cos’è QualCoder

QualCoder è un software desktop gratuito e open source per l’analisi dei dati qualitativi.

QualCoder consente di codificare testi, immagini, audio e video, nonché di redigere appunti di diario e promemoria. È possibile organizzare i codici in uno schema di categorizzazione ad albero. È possibile generare vari report. La maggior parte dei report può essere esportata in formato HTML, Open Document Text (ODT), testo semplice o file Excel.

QualCoder include una serie di funzionalità assistite dall’intelligenza artificiale che utilizzano modelli linguistici di grandi dimensioni per aiutare a esplorare, analizzare e interagire con i dati in modi innovativi. Prendi familiarità con i nuovi dati esplorando argomenti o concetti generali in una chat interattiva con l’IA. Approfondisci aspetti specifici grazie alla codifica e all’analisi testuale assistite dall’IA. E se arrivi al punto in cui devi sintetizzare e consolidare i tuoi risultati, discutili in una chat sui codici con l’IA.

!!! note "Perché utilizzare QualCoder?"
    - **QualCoder è gratuito**. Molti software di analisi qualitativa richiedono costose spese una tantum o abbonamenti mensili.
    - **QualCoder è facile da usare**. Ha tutto ciò che serve per eseguire analisi qualitative senza le interfacce complicate di alcune alternative
    - **QualCoder funziona offline**. Internet non è sempre disponibile e QualCoder non richiede una connessione a Internet per funzionare.
    - **QualCoder non è vincolato a un computer**. Se cambi posto di lavoro, non devi preoccuparti di rimanere vincolato alla licenza del tuo precedente posto di lavoro né di acquistarne una nuova. La licenza di QualCoder ti consente di utilizzare il software indipendentemente da dove lavori o su quale computer sia installato.
    - **QualCoder è multipiattaforma**. Funziona su Linux, Windows e macOS; ciò significa che non devi preoccuparti se cambi sistema operativo e che puoi collaborare con colleghi che utilizzano piattaforme diverse. Vedi la nota.
    - **QualCoder si affida alla comunità**. Se trovi un errore, hai una richiesta di funzionalità o un feedback, segnalalo sulla [pagina di QualCoder su GitHub](https://github.com/ccbogel/QualCoder).
    - **QualCoder è in continuo miglioramento**. QualCoder è in fase di sviluppo attivo, il che significa che vengono rilasciate versioni sempre più recenti e migliorate.
    - **QualCoder supporta gli standard aperti**. QualCoder mira a supportare lo [standard REFI-QDA](https://www.qdasoftware.org/); è possibile scambiare codebook e progetti con i propri colleghi anche se questi non utilizzano QualCoder, purché il software da loro utilizzato supporti lo standard REFI-QDA. Ciò significa che non si corre il rischio che i propri dati diventino inaccessibili. La conformità al progetto REFI-QDA non è ancora garantita al 100%. Sono necessari ulteriori test, in particolare per l’esportazione e l’importazione di file audio e video e per i file collegati in modo relativo.
    - **QualCoder può essere modificato**. È possibile modificare e adattare QualCoder alle proprie esigenze, a condizione che le modifiche apportate vengano rese disponibili a tutti. Ciò significa anche che è possibile copiare il software e distribuirlo gratuitamente ai propri colleghi o studenti.
    - **QualCoder è una soluzione software a sostegno della scienza aperta**. [Raccomandazioni dell’UNESCO sulla scienza aperta](https://www.unesco.org/en/open-science?hub=686)
