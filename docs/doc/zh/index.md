# 什么是 QualCoder

QualCoder 是一款用于定性数据分析的免费开源桌面软件。

QualCoder 支持对文本、图像、音频和视频进行编码，并可撰写日记笔记和备忘录。您可以按照树状分类方案对编码进行整理。该软件可生成多种报告，其中大部分报告均可导出为 HTML、开放文档文本（ODT）、纯文本或 Excel 文件。

QualCoder 包含一系列人工智能辅助功能，这些功能利用大型语言模型，以创新的方式帮助您探索、分析和与数据互动。您可以通过与 AI 进行交互式对话，探索广泛的主题或概念，从而熟悉新数据。借助 AI 辅助编码和文本分析，您可以更深入地探究特定方面。当您需要综合和整合分析结果时，还可以在代码聊天中与 AI 进行讨论。

!!! note "为什么选择 QualCoder"
    - **QualCoder 完全免费**。许多定性分析软件都需要支付昂贵的一次性费用或月费。
    - **QualCoder 易于使用**。它具备进行定性分析所需的一切功能，却没有某些同类软件那样的复杂界面
    - **QualCoder 支持离线使用**。网络并非随时可用，而 QualCoder 运行无需网络连接。
    - **QualCoder 不受电脑限制**。若您更换工作场所，无需担心受限于原工作场所的许可证，也无需购买新许可证。QualCoder 许可证允许您在任何工作地点、任何安装该软件的电脑上使用。
    - **QualCoder 支持多平台**。它可在 Linux、Windows 和 macOS 上运行，这意味着您无需担心更换操作系统，同时也意味着您可以与使用不同平台的同事进行协作。参见注释。
    - **QualCoder 依赖于社区**。如果您发现错误、有功能请求或反馈，请在 [QualCoder 的 GitHub 页面](https://github.com/ccbogel/QualCoder) 上留言。
    - **QualCoder 持续改进**。QualCoder 正在积极开发中，这意味着会不断发布更新、更完善的版本。
    - **QualCoder 支持开放标准**。QualCoder 致力于支持 [REFI-QDA 标准](https://www.qdasoftware.org/)，即使您的同事未使用 QualCoder，只要他们使用的软件支持 REFI-QDA 标准，您仍可与他们交换编码本和项目。这意味着您的数据不会面临无法访问的风险。目前尚不能保证 QualCoder 项目 100% 符合该标准。还需要进行进一步测试，特别是针对音频和视频的导出与导入，以及相对链接文件。
    - **QualCoder 支持修改**。您可以根据自身需求修改和调整 QualCoder，只要您将修改内容公开发布供所有人使用即可。这也意味着您可以复制该软件，并免费将其提供给您的同事或学生。
    - **QualCoder 是一款支持开放科学的软件选项**。[联合国教科文组织关于开放科学的建议](https://www.unesco.org/en/open-science?hub=686)
