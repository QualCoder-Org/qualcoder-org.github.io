# Paramètres

La boîte de dialogue des paramètres permet de modifier plusieurs fonctionnalités, notamment la langue, le nom du codeur⋅euse, le type et la taille de la police, ainsi que les options d'intégration de l'IA.

### Nom de codeur⋅euse
Plusieurs codeur⋅euses peuvent coder et annoter le même texte. Votre nom de codeur⋅euse est affiché au dessus de la fenêtre de configuration. Votre nom est utilisé pour identifier vos contribution au projet. Cliquez sur le bouton « ... » pour changer votre identité ou créer, renommer ou fusionner d’autres codeurs au sein du projet courant. Vous pouvez également gérer ici la visibilité des autres collaborateurs. La fenêtre est également accessible depuis la barre d’outils dans les espaces de code de texte, images et PDF.

### Configuration générale

La langue par défaut de QualCoder est l'anglais. Actuellement, le français, l'allemand, l'italien, le portugais, l'espagnol et le suédois sont disponibles comme langues alternatives. Ces traductions n'ont pas été entièrement vérifiées pour leur exactitude. Lorsque vous changez de langue, vous devez fermer puis rouvrir QualCoder pour que le changement prenne effet.

Cochez la case Afficher les ID pour afficher les ID numériques des fichiers de codes, des cas et plus encore. Ces ID peuvent être utiles lors de la rédaction de requêtes SQL.

Il existe quelques options permettant de modifier l'apparence générale de QualCoder.

Pour la transcription audio et vidéo, vous pouvez sélectionner le format horaire et le type de parenthèses.

Les fichiers texte très volumineux peuvent ralentir le mode de codage dans QualCoder. Une option accessible par un clic droit dans Code Text permet donc de charger des morceaux de texte de taille réduite. La taille par défaut est de 50 000 caractères, mais vous pouvez la réduire dans les paramètres pour les ordinateurs moins performants.

### Sauvegardes

La sauvegarde du projet peut être configurée pour s'effectuer automatiquement toutes les heures, ou pas du tout. QualCoder conserve les sauvegardes les plus récentes avec un horodatage de date et d'heure. Le nom de la sauvegarde est : _nomduprojet_BKP_aaaammjj_hh.qda_, où _hh_ correspond à l'heure en format 24 heures. Cela signifie également qu'aucune sauvegarde supplémentaire n'est effectuée dans la même heure. Un nombre limité de sauvegardes est conservé, **les sauvegardes les plus anciennes sont supprimées**. L'option de paramétrage permet de choisir entre 1 et 10 sauvegardes.

Il est recommandé de faire une sauvegarde avant d'effectuer des modifications importantes, comme la réorganisation des codes et des catégories, la fusion de projets ou de codes. QualCoder effectue des sauvegardes horaires (en conservant jusqu'à cinq sauvegardes récentes) si cette option est activée dans les paramètres.

Renommez ou déplacez vers un autre emplacement les sauvegardes spécifiques que vous souhaitez conserver. Lorsque les sauvegardes sont activées, une option supplémentaire permet de choisir de sauvegarder également les fichiers audio/vidéo. Pour des sauvegardes complètes, il est recommandé de sauvegarder les fichiers audio et vidéo, bien que cela puisse ralentir l'ouverture du logiciel si vous avez de nombreux fichiers volumineux. Alternativement, vous pouvez lier les fichiers audio/vidéo volumineux plutôt que de les importer dans le dossier du projet. Le répertoire par défaut du projet est celui vers lequel de nombreuses opérations de fichiers pointeront par défaut, pour l'export ou l'import de fichiers.

Dans votre dossier personnel, QualCoder créera un dossier `.qualcoder` contenant un fichier `QualCoder.log` pour enregistrer les événements et un fichier `config.ini` qui contiendra de nombreux paramètres, tels que le nom du codeur⋅euse actuel, les polices préférées, la langue et un répertoire de travail préféré. Ne stockez aucune donnée ou projet dans le dossier `.qualcoder`. Le seul fichier que vous pouvez y placer est un fichier `stopwords.txt`, uniquement si vous souhaitez appliquer un ensemble différent de mots vides pour les fonctions de nuage de mots et de comptage de mots. Ce dossier est réservé à l'usage interne de QualCoder pendant son exécution.
