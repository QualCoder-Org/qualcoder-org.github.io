# Organisateur de code

L'organisateur de code vous permet de modifier la structure de l'arborescence de codage.

Lors de la première ouverture, vous serez invité à sauvegarder votre projet avant d'effectuer et d'appliquer des modifications. Les modifications effectuées ici ne peuvent pas être annulées.

![image](https://github.com/user-attachments/assets/ae81b366-54e8-457f-9afe-abb7f76d2a41)

À l'aide du bouton situé à l'extrême gauche, sélectionnez toutes les branches de code de votre projet ou une branche de code spécifique uniquement.

![image](https://github.com/user-attachments/assets/57b82319-1145-4872-a6c7-63cbc807b370)

Toutes les sous-catégories et tous les codes de votre sélection s'affichent.

![image](https://github.com/user-attachments/assets/303ddd26-605c-4963-9db4-ba7c50c40250)

**Codes :**

Cliquez-droit sur un code pour obtenir les options suivantes : supprimer de la catégorie, déplacer vers une autre catégorie, fusionner avec un autre code, afficher/modifier la note, renommer, afficher le média codé, afficher la note sous le nom du code.

**Catégories :**

Un bouton situé en haut de l'écran permet d'ajouter une nouvelle catégorie. 

Cliquez-droit sur une catégorie pour : fusionner une catégorie avec une autre, déplacer une catégorie vers une autre, supprimer une catégorie d'une autre, renommer une catégorie, afficher et modifier la note, ou simplement afficher la note sous le nom de la catégorie.

Affichage de la note sous le nom de la catégorie

![image](https://github.com/user-attachments/assets/92fee31b-1b70-4bdf-bb15-51b0485e4739)

Un bouton permet d'exporter l'image au format png.

Le bouton Appliquer permet d'appliquer les modifications à la structure arborescente du codage. Si vous ne cliquez pas sur ce bouton, les modifications ne seront pas prises en compte.
