# Attributs

Les attributs sont des variables associées à des fichiers, des cas ou des journaux. Ils peuvent être utiles pour apporter du contexte lors de l'interprétation de votre analyse. Ouvrez la boîte de dialogue « Gérer les attributs ». Vous pouvez ajouter, supprimer, renommer des attributs et y ajouter des notes.

Vous pouvez cliquer avec le bouton droit sur un attribut numérique pour le convertir en chaîne de caractères.

Les seules options disponibles pour les attributs sont « Numérique » et « Chaîne de caractères ». Il n'y a pas d'options pour les types booléens, de date ou d'heure, ni pour des types numériques spécifiques tels que les entiers.  Toutefois, si vous exécutez des requêtes SQL, vous devrez peut-être convertir l'attribut numérique en entier ou en nombre réel : cast(my_var as integer).

Les attributs peuvent être ajoutés dans les sections Gérer les fichiers, Gérer les cas ou Gérer les journaux en cliquant sur le bouton (x). 

Les attributs peuvent être importés à partir d’une enquête, ainsi qu’à partir d’un fichier de projet REFI-QDA. Remarque : l’importation d’un projet REFI-QDA n’est pas garantie.

QualCoder n’est pas conçu pour effectuer des analyses statistiques des attributs. Il est préférable d’utiliser un logiciel statistique tel que [R](https://www.r-project.org).
