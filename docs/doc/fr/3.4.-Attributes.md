# Attributs

Les attributs sont des variables associées à des fichiers, des cas ou des journaux. Ils peuvent être utiles pour ajouter du contexte lors de l'interprétation de votre analyse. Ouvrez la boîte de dialogue Gérer les attributs. Vous pouvez ajouter, supprimer, renommer et ajouter des notes aux attributs.

Vous pouvez cliquer avec le bouton droit sur un attribut numérique pour le convertir en caractère.

Les seules options disponibles pour les attributs sont les types numériques et caractères. Il n'y a pas d'options pour les types booléens, de date ou d'heure ou numériques spécifiques tels que les entiers.  Cependant, si vous exécutez des requêtes SQL, vous devrez peut-être convertir l'attribut numérique en entier ou en réel : cast(my_var as integer).

Les attributs peuvent être ajoutés dans les sections Gérer les fichiers, Gérer les cas ou Gérer les journaux en cliquant sur le bouton (x). 

Les attributs peuvent être importés à partir d'une enquête, mais aussi à partir d'un fichier de projet REFI-QDA (REFI-QDA est expérimental).

QualCoder n'est pas destiné à effectuer des analyses statistiques des attributs. Il est préférable d'utiliser un logiciel statistique dédié tel que [R](https://www.r-project.org/) ou tout autre logiciel similaire.

Les attributs peuvent également être ajoutés ou supprimés à l'aide de l'option Gérer > Gérer les attributs :

![image](https://github.com/user-attachments/assets/cb9695f5-acce-406b-a8da-07c8b3273752)

Dans cette zone, vous pouvez supprimer des attributs. Cependant, même si vous pouvez supprimer des attributs de référence, nous vous recommandons de ne pas le faire, sauf si vous êtes certain que toutes vos références sont inutiles. Ceux-ci sont préfixés par « Ref_ ».
