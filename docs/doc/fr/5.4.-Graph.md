# Graphique

Un graphique des catégories et des codes s'affiche. Il existe des options permettant de limiter les codes et les catégories affichés, telles que « Toutes les catégories » ou « Catégories sélectionnées ». Cliquez sur la zone d'image et sur + ou – pour zoomer ou dézoomer.

Chaque code ou catégorie peut être déplacé en cliquant près du bord de la case et en le faisant glisser. En cliquant sur le texte, vous pouvez modifier temporairement le libellé du code ou de la catégorie. Un clic droit sur un code affiche un menu d'options telles que l'affichage de la note ou l'affichage de tout le texte du dossier ou du fichier qui a été codé avec le code sélectionné.

Un clic droit sur une ligne vous permet de modifier son épaisseur, de passer à un style de ligne pointillée ou de changer la couleur de la ligne. Cela peut être utile pour mettre l'accent sur certains éléments. Un clic droit sur un espace vide vous permet d'ajouter des zones de texte libres, des zones de texte de fichiers et des zones de texte de cas, ainsi que d'insérer des lignes supplémentaires entre les composants.

Le texte codé, les images codées et les segments audiovisuels codés peuvent être affichés. Les mémos du texte codé, des images codées et des segments audiovisuels codés peuvent être affichés dans des zones de texte. Chacun d'entre eux dispose d'infobulles pour fournir plus d'informations. Vous ne pouvez pas afficher plusieurs fois le même segment codé dans le graphique.

Les graphiques peuvent être enregistrés, chargés et supprimés. Les graphiques peuvent être exportés sous forme de fichiers png.
