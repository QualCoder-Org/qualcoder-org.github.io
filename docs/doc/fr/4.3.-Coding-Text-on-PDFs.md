# Codage des PDF

Les PDF peuvent être affichés et leur texte peut être codé. Les PDF volumineux prendront plus de temps à charger.

Les PDF affichés avec QualCoder ne sont pas des répliques exactes, ils sont affichés avec certaines limitations.

L'écran est divisé en trois volets. Le volet de gauche contient la liste des fichiers et l'arborescence des codes. Le volet central affiche la représentation PDF. Le volet de droite affiche le texte brut extrait du PDF.

## Taille de la police et zoom

Les polices d'origine du texte ne sont pas utilisées (pour diverses raisons, notamment parce qu'il est trop difficile d'extraire les polices et de les utiliser à partir des PDF). À la place, la police standard utilisée dans QualCoder est utilisée dans les zones de texte PDF affichées. Une liste déroulante permet de réduire la taille de la police afin d'éviter le chevauchement du texte. Elle est réglée sur -2, mais -3 peut également convenir. ## L'extrait de texte brut doit correspondre

Le texte brut du PDF qui est chargé lors de l'importation d'un PDF via Gérer les fichiers doit correspondre exactement au texte brut qui est rechargé lorsque le PDF est chargé pour afficher le PDF ici. Cela signifie que les PDF chargés avant la version 3.4 ne correspondront pas correctement. Il est donc préférable de ne faire qu'afficher le PDF et de ne pas le coder ici. La sélection des zones de texte à coder est désactivée si les longueurs de texte ne correspondent pas. De même, si le texte brut est modifié de quelque manière que ce soit et que les longueurs de texte ne correspondent pas, le codage via la sélection des zones de texte PDF est désactivé. L'affichage des bandes de codage dans la  page PDF  présentée peut également sembler incorrect.

## Fonctions supplémentaires 

Plusieurs boutons sont disponibles en haut de la fenêtre. L'un d'eux permet d'exporter une image de la page PDF. Le bouton en forme de loupe ouvre une fenêtre qui affiche les éléments PDF utilisés sur la page.

En haut de la fenêtre se trouvent également plusieurs cases à cocher permettant d'afficher ou de masquer les éléments PDF du texte et des images. Une case à cocher affichera tout le texte en noir, ce qui peut être utile lorsque du texte de couleur claire est affiché sur un fond clair.

## Volet droit

Le volet droit affiche le texte brut tel qu'il serait affiché dans le texte du code. Vous pouvez marquer, décocher, ajouter des notes au texte codé, marquer le codage comme important et annoter le texte ici.

## Volet central

Pour coder, cliquez avec le bouton gauche de la souris et faites glisser la souris pour sélectionner les zones de texte du PDF. Cela peut s'avérer difficile selon la taille et la position des zones. Cliquez ensuite sur un code pour attribuer la sélection au code. D'autres ajustements peuvent être effectués dans le volet de texte brut de droite à l'aide des flèches gauche ou droite pour agrandir, ou des flèches gauche et droite avec la touche Alt enfoncée pour réduire la section de texte codé.

Cliquez-droit de la souris sur un segment codé pour afficher le menu contextuel. Ce menu vous permet d'ajouter une note, de marquer comme important ou de décocher le segment codé.

### Raccourcis

H Afficher ou masquer les sections supérieures de l'écran.

Ctrl F Aller à la zone de recherche de texte.

Ctrl Z Annuler le dernier code non marqué.

Ctrl 1 Aller au fichier suivant.

Ctrl 2 Aller au dernier fichier codé.

Ctrl 4 Afficher et modifier la note du fichier.

Ctrl 5 Obtenir la liste des fichiers à partir des attributs.

Ctrl 8 

Ctrl 9 Afficher les segments codés importants. Active ou désactive cette fonction.

Ctrl 0 Affiche la page d'aide dans le navigateur.

#### Dans la zone d'affichage du PDF :

Vous pouvez zoomer et dézoomer en cliquant sur le PDF affiché et en appuyant sur les touches + ou -.
Appuyez sur Q pour marquer rapidement la sélection avec le code récent.
Appuyez sur R pour marquer rapidement la sélection avec un choix de codes récemment utilisés.

#### Dans la zone de texte brut :

A Annoter - pour la sélection actuelle - édition de texte uniquement

Q Marquer rapidement avec un code - pour la sélection actuelle

I Marquer comme important

M Code mémo - à la position cliquée

O Raccourci pour parcourir les codes qui se chevauchent - à la position cliquée

R Ouvre un menu contextuel pour les codes récemment utilisés pour marquer du texte

U Démarquer à l'emplacement sélectionné

V Attribuer le code « in vivo » au texte sélectionné
