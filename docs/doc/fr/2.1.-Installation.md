# Instructions d'installation et d'exécution de QualCoder

## Installation

Pour les installations manuelles (via la ligne de commande ou le terminal), nous mentionnons la dernière version à jour. Veuillez vérifier sur la [page des versions](https://github.com/ccbogel/QualCoder/releases) pour obtenir la version la plus récente, qui peut être plus à jour.

### Prérequis
VLC est optionnel, mais nécessaire si vous souhaitez effectuer du codage audio/vidéo. Optionnellement, sur certains systèmes d’exploitation (comme Fedora), QualCoder ne peut pas trouver VLC. Optionnellement, installez ffmpeg pour la conversion parole-texte et la visualisation du signal, voir [phoenixnap.com/kb/ffmpeg-windows](https://phoenixnap.com/kb/ffmpeg-windows).

=== "Windows"

    Prenez la [dernière version](https://github.com/ccbogel/QualCoder/releases/latest).
    
    Vous avez deux options pour les exécutables, disponibles pour Windows 11 :
    
    Option 1 : Il y a un installateur Windows disponible, intitulé : **QualCoder.Win.exe**
    
    Option 2 : Une version portable est disponible sous forme de dossier zip : **QualCoder.Win.Portable.zip**. Téléchargez et décompressez-le. Le fichier exécutable se trouve dans un dossier, accompagné d'un sous-dossier interne. Ces deux éléments doivent rester ensemble pour que l'exécutable fonctionne. Cliquez sur l'exécutable QualCoder pour le lancer (le démarrage peut prendre jusqu'à 20 secondes).
    
    Version portable autonome : <img width="100" height="60" alt="QualCoder portable" src="https://github.com/user-attachments/assets/6be1de98-8b47-4d57-b4b5-c34375e44959" />
    
    Lors de la première utilisation, Windows peut vous demander d'autoriser l'exécution de QualCoder, car il provient d'un éditeur inconnu. Obtenir un certificat d'éditeur de confiance coûte cher, ce qui n'est pas envisageable pour l'instant. Si ces avertissements vous gênent, installez-le à partir du code source comme détaillé ci-dessous.
    
    De manière alternative, vous pouvez installer depuis le code source :
    
     Utilisez un environnement virtuel (commandes indiquées au point 6 ci-dessous). Si vous n'utilisez pas d'environnement virtuel, cela pourrait affecter d'autres logiciels Python que vous avez éventuellement installés.

    1. Téléchargez et installez le langage de programmation Python. Veuillez utiliser une version récente sous Windows.  [Python3](https://www.python.org/downloads/). Téléchargez le dernier « programme d'installation Windows (64 bits) » (ou celui correspondant à votre architecture) pour l'une des versions de Python mentionnées ci-dessus.

    IMPORTANT : dans la première fenêtre de l’installation, cochez l’option « Ajouter Python au PATH ».

    2. Téléchargez le logiciel QualCoder à partir de : https://github.com/ccbogel/QualCoder en cliquant sur le bouton vert « Code ». Il s’agit de la version la plus récente, mais qui n’a pas encore été officiellement publiée (des erreurs de code peuvent parfois s’y glisser).  Cliquez sur le bouton vert « Code », puis sur « Télécharger le fichier ZIP ». **Sinon**, choisissez le fichier ZIP de la version la plus récente ; vous trouverez le lien vers les versions (Releases) sur le côté droit de cette page.

    3. Décompressez le dossier à l’emplacement de votre choix (par exemple, le dossier « Téléchargements »). (Astuce : supprimez le double nom du dossier de décompression « QualCoder-master\QualCoder-master » lorsqu’on vous demande où décompresser. Indiquez simplement « QualCoder-master »). 

    4. Utilisez l’invite de commande Windows. Tapez « cmd » dans le moteur de recherche du menu Démarrer de Windows, puis cliquez sur l’icône noire « cmd.exe » — la console de commande de Windows. Dans la console, tapez ou collez la commande en utilisant le copier-coller par clic droit (Ctrl+V ne fonctionne pas).

    5. Dans l’invite de commande, accédez (à l’aide de la commande `cd`) au dossier QualCoder. Vous devriez vous trouver dans le dossier QualCoder-master ou, si vous utilisez une version stable, dans le dossier QualCoder-3.8.2. Par exemple : 

    ```bash
    cd Téléchargements\QualCoder-master
    ```

    5a. Au lieu d’exécuter les commandes ci-dessous, lancez le fichier .bat qui exécutera toutes ces commandes en tapant ceci puis en appuyant sur Entrée : 

    `run_from_source_Windows.bat`

    Si vous rencontrez des erreurs en utilisant un fichier .bat, exécutez plutôt les commandes ci-dessous.

    6. Installez l’environnement virtuel et les modules Python requis. 



    L’installation peut prendre jusqu’à 10 minutes. Sur certains systèmes Windows, vous devrez peut-être remplacer la commande _python_ par _py_ ou _python3_ ci-dessous : 

    ```bash
    python -m venv env
    env\Scripts\activate
    python -m pip install --upgrade pip
    python -m pip install -r requirements.txt
    ```

    7. Lancez QualCoder en vous plaçant d’abord dans le dossier src :

    ```bash
    cd src
    python -m qualcoder
    ```


    8. Si vous exécutez QualCoder dans un environnement virtuel, pour quitter cet environnement, tapez :

    `deactivate`

    L'invite de commande supprimera alors la mention *(env)*.

    **Pour relancer QualCoder**

    Si vous n'utilisez pas d'environnement virtuel, tant que vous vous trouvez sur la même lettre de lecteur, par exemple C:

    `python -m qualcoder`

    Si vous utilisez un environnement virtuel :

    utilisez la commande `cd` pour accéder au dossier src situé dans le dossier Qualcoder-master (ou dans le dossier de la version de Qualcoder) :

    ```
    env\Scripts\activate
    cd src
    python -m qualcoder
    ``` 

=== "macOS"

    **Installation depuis l’App bundles :**
    
    Assurez-vous d'avoir installé toutes les mises à jour macOS avant de commencer. Allez dans l'application Réglages système de votre Mac. Cliquez sur Général dans la barre latérale (il peut être nécessaire de faire défiler), puis sur Mise à jour logicielle.
    
    Pour installer QualCoder : Double-cliquez sur le fichier .dmg téléchargé. Glissez QualCoder vers le lien de votre dossier Applications.
    
    Lancez QualCoder depuis le Launchpad ou en double-cliquant sur l'application dans votre dossier Applications.
    Nous ne pouvons pas actuellement signer les bundles d'application. Lors du premier lancement, un message peut apparaître indiquant que QualCoder provient d'un développeur non enregistré ou que "QualCoder ne peut pas être ouvert car Apple ne peut pas vérifier l'absence de logiciels malveillants". Selon votre processeur, vous verrez l'une de ces options :
    
    - Si vous voyez « Afficher dans le Finder » ou OK, cliquez sur OK.
    - Si vous voyez « Terminé » ou « Placer dans la corbeille », cliquez sur « Terminé ».
    
    Ensuite, allez dans Réglages > Confidentialité et sécurité. Faites défiler jusqu'à voir un message indiquant que QualCoder a été empêché de s'ouvrir. Cliquez sur « Ouvrir quand même ». Vous devrez peut-être entrer votre mot de passe et cliquer à nouveau sur "Ouvrir". Après avoir entré votre mot de passe, attendez quelques secondes que le programme se lance.
    
    Désormais, QualCoder devrait démarrer sans problème.
    
    Vous pouvez installer QualCoder sans les droits admin. Vous devez copier le binaire  dans  le dossier /Users/mylogin/Applications. Vous devez supprimer l’attribut dans le répertoire de l’application, en tapant dans le terminal :
    
    ```xattr -d com.apple.quarantine /Users/mylogin/Applications/QualCoder.app```
    
    
    **Installation à partir du code source :**
    
    Utilisez un environnement virtuel (commandes indiquées au point 6 ci-dessous). Ne pas utiliser d'environnement virtuel pourrait affecter d'autres logiciels Python que vous auriez installés.

    1. Téléchargez et installez le langage de programmation Python. Veuillez utiliser Python 3.10, 3.11 ou 3.12 sous Windows ; d'autres versions pourraient poser des problèmes  [Python3](https://www.python.org/downloads/macos/). Téléchargez le dernier « programme d’installation macOS 64 bits universal2 » correspondant à l’une des versions de Python mentionnées ci-dessus, puis ouvrez-le pour installer Python.

    2. Téléchargez le logiciel QualCoder à l’adresse suivante : https://github.com/ccbogel/QualCoder en cliquant sur le bouton « Green Code ». Il s’agit de la version la plus récente, mais qui n’a pas encore été officiellement publiée (des erreurs de code peuvent parfois s’y glisser).  Cliquez sur le bouton vert « Code », puis sur « Télécharger le fichier ZIP ». **Sinon**, choisissez le fichier ZIP de la version la plus récente ; vous trouverez le lien vers les versions dans la partie droite de cette page.

    3. Décompressez le dossier à l'emplacement de votre choix (par exemple, le dossier « Téléchargements ») en double-cliquant dessus. 

    4. Utilisez l'application Terminal (`Applications -> Utilitaires -> Terminal`).

    5. Dans le Terminal, accédez (à l’aide de la commande `cd`) au dossier QualCoder. Vous devriez vous trouver dans le dossier QualCoder-master ou, si vous utilisez une version publiée (par exemple, le dossier QualCoder-3.8.2). Par exemple : 

    ```bash
    cd Téléchargements/QualCoder-master
    ```

    6. Installez l’environnement virtuel et les modules Python requis. 

    La commande `python3` utilise la version la plus récente de Python installée. Vous pouvez utiliser une version spécifique sur votre macOS si plusieurs versions de Python sont installées, par exemple `python3.10`. Pour vérifier que vous utilisez la bonne version de Python, tapez `which python3`, ce qui devrait afficher : `/Library/Frameworks/Python.framework/Versions/3. <version>/bin/python3`. Si la sortie est `/usr/bin/python3`, ne continuez pas, car il s’agit du Python de votre système et son utilisation est déconseillée.

    L’installation peut prendre jusqu’à 10 minutes. 

    ```bash
    python3 -m venv env # cela crée l’environnement virtuel nommé « env » dans votre répertoire actuel
    source env/bin/activate # cela active l'environnement virtuel « env » ; (env) devrait apparaître devant votre invite de commande
    pip3 install --upgrade pip # facultatif ; pip et pip3 sont équivalents au sein d'un environnement virtuel
    pip3 install -r requirements.txt
    ```

    7. Lancez QualCoder depuis l'invite de commande

    ```bash
    cd src
    python3 -m qualcoder # python et python3 sont équivalents au sein d’un environnement virtuel
    ```

    8. Si vous exécutez QualCoder dans un environnement virtuel, pour quitter celui-ci, tapez :

    ```bash
    deactivate
    ```

    L’invite de commande supprimera alors la mention *(env)*.

    **Pour redémarrer QualCoder**

    Si vous n'utilisez pas d'environnement virtuel :

    ```bash
    cd Téléchargements/QualCoder-master
    cd src 
    python3 -m qualcoder
    ```

    Si vous utilisez un environnement virtuel :

    ```bash
    cd Téléchargements/QualCoder-master
    source env/bin/activate
    cd src 
    python3 -m qualcoder
    ```


=== "Debian / Ubuntu / Lubuntu / ZorinOS Linux"

    Un fichier exécutable est disponible sur la page des versions pour Ubuntu. Téléchargez-le et double-cliquez pour l'exécuter.
    
    **Exécution à partir du code source :**
    Pour effectuer l'installation à partir du code source ci-dessous, dans un environnement virtuel. Si vous utilisez **Xfce**, le gestionnaire de bureau alternatif d'Ubuntu, vous devrez peut-être exécuter la commande suivante : `sudo apt install libxcb-cursor0`. Si vous utilisez des fichiers audio ou vidéo, installez VLC (à télécharger depuis le site) ou : `sudo apt install vlc`

    0. Installez pip. Il s'agit d'un outil permettant de télécharger des modules Python supplémentaires.

    `sudo apt install python3-pip`

    1. Téléchargez et décompressez le dossier QualCoder. 

    2. Accédez ensuite au dossier QualCoder à l'aide de la commande `cd`.

    Par exemple, vous vous trouvez peut-être actuellement dans ce dossier, où vous avez décompressé QualCoder : 

    votrecomputer:~Téléchargements/QualCoder-3.8.2

    3a. Au lieu des commandes à partir du point 3b., exécutez le fichier shell qui lancera toutes les commandes ci-dessous. Rendez ce fichier exécutable (clic droit puis Propriétés), puis tapez la commande ci-dessous et appuyez sur Entrée : 

    `./run_from_source_Linux.sh`

    3b. Configurez l’environnement virtuel et installez les modules Python. L’environnement virtuel se trouvera dans son propre dossier nommé « env ». L’installation des modules requis prend un certain temps.

    ```
    python3 -m venv env
    source env/bin/activate
    pip install --upgrade pip
    pip install -r requirements.txt
    ```

    5. Accédez au dossier « src » situé à l’intérieur, puis :
    ```
    cd src
    python3 -m qualcoder
    ```

    6. Après avoir utilisé QualCoder, désactivez l’environnement virtuel.

    `deactivate`

    **À tout moment après l’installation, accédez au dossier QualCoder puis :**

    ```
    source env/bin/activate
    cd src
    python3 -m qualcoder
    ```

    Pour quitter l’environnement :

    `deactivate`

    Vous pouvez également créer un fichier .desktop pour lancer QualCoder :

    Créez un fichier .desktop pour le lancement, entrez cette commande (adaptez-la en fonction de l'emplacement du dossier du code source) :

    bash -c 'cd ~/.local/share/qualcoder/src/ && ~/.local/share/qualcoder/env/bin/python3.12 -m qualcoder'

=== "Fedora Linux"

    Ces instructions permettent de télécharger le code source actuel directement depuis GitHub. Remarque : Fedora utilise Wayland, qui peut ne pas fonctionner correctement avec l'interface graphique Qt. Il est recommandé d'installer également Xwayland.
    
    **Il existe un problème récurrent avec le codage audio/vidéo sous Fedora : le logiciel peut planter.**

  
    1. Téléchargez et décompressez le dossier Qualcoder. 

    2. Accédez ensuite au dossier QualCoder à l’aide de la commande `cd`.

    Par exemple, vous vous trouvez peut-être actuellement dans ce dossier, où vous avez décompressé QualCoder : 

    votrecomputer:~Téléchargements/QualCoder-3.8.2

    3a. Au lieu des commandes à partir du point 3b., exécutez le fichier shell qui lancera toutes les commandes ci-dessous. Rendez ce fichier exécutable (clic droit puis sélectionnez « Propriétés »), puis tapez la commande ci-dessous et appuyez sur Entrée : 

    `./run_from_source_Linux.sh`

    3b. Configurez l’environnement virtuel et installez les modules Python. L’environnement virtuel se trouvera dans son propre dossier nommé « env ». L’installation des modules requis prend un certain temps.

    ```
    python -m venv env
    source env/bin/activate
    pip install --upgrade pip
    pip install -r requirements.txt
    ```

    3. Accédez au dossier src :
       
    ```
    cd src
    ```

    4. Lancez ensuite QualCoder :

    ```
    python -m qualcoder
    ```

    5. Après avoir utilisé QualCoder, désactivez l’environnement virtuel.

    `deactivate`

    6. Pour l’utiliser à tout moment après l’installation, accédez au dossier QualCoder, puis :

    ```
    source env/bin/activate
    cd src
    python3 -m qualcoder
    ```

    Une fois terminé, tapez `deactivate` pour quitter l’environnement virtuel.
