# QualCoder Installation und Start

## INSTALLATION 

Bei den manuellen Installationen (über die Befehlszeile oder das Terminal) beziehe ich mich auf eine bestimmte Version, zum Beispiel Version 3.7. Bitte prüfen Sie auf der [Releases-Seite](https://github.com/ccbogel/QualCoder/releases) die aktuellste Version, da diese neuer sein kann.

### Voraussetzungen
`VLC` ist optional, wird aber benötigt, wenn Sie Audio- oder Video-Coding durchführen möchten. Auf einigen Betriebssystemen (z. B. Fedora) findet QualCoder `VLC` gelegentlich nicht automatisch. Optional können Sie `ffmpeg` für die Wellenform-Anzeige (Waveform) installieren, siehe [phoenixnap.com/kb/ffmpeg-windows](https://phoenixnap.com/kb/ffmpeg-windows).

=== "Windows"

    Wählen Sie das [neueste Release](https://github.com/ccbogel/QualCoder/releases/latest).
    
    Für Windows 11 gibt es zwei Möglichkeiten:
    
    Option 1: Ein Windows-Installer auf der Release-Seite namens: `QualCoder.Win.exe`
    
    Option 2: Eine portable Version (ohne Installation) namens: `QualCoder.Win.Portable.zip`. Laden Sie diese herunter und entpacken Sie sie. Die `exe`-Datei befindet sich in einem Ordner zusammen mit einem internen Unterordner. Beide müssen zusammenbleiben, sonst startet das Programm nicht. Klicken Sie doppelt auf die `QualCoder.exe`, um sie zu starten; es kann bis zu 20 Sekunden dauern, bis das Programm öffnet.
    
    Portable Version: <img width="100" height="60" alt="QualCoder portable" src="https://github.com/user-attachments/assets/6be1de98-8b47-4d57-b4b5-c34375e44959" />
    
    Beim ersten Start fragt Windows, ob Sie das Ausführen von QualCoder zulassen möchten. Das liegt daran, dass die Software von einem unbekannten Herausgeber stammt. Ein Zertifikat für einen vertrauenswürdigen Herausgeber ist sehr teuer, weshalb dies vorerst nicht möglich ist. Wenn Sie diese Warnungen vermeiden möchten, installieren Sie die Software aus dem Quellcode, wie im nächsten Abschnitt beschrieben.
    
    Alternativ Installation aus dem Quellcode:
    
    Laden Sie den Quellcode herunter. Lesen und befolgen Sie den Abschnitt für die Windows-Installation in der Datei `Readme.md`. Entfernen Sie die Backticks ( ` ), die die Befehle umschließen, da diese nur zur Darstellung dienen.

=== "MacOS"
    
    **Installation über App-Bundles:**
    
    Stellen Sie sicher, dass Ihr macOS aktuell ist. Öffnen Sie die „Systemeinstellungen“, klicken Sie in der Seitenleiste auf „Allgemein“ und dann auf „Softwareupdate“.
    
    QualCoder installieren: Klicken Sie doppelt auf die heruntergeladene `dmg`-Datei und ziehen Sie QualCoder in Ihren Programme-Ordner.
    
    Starten Sie QualCoder über das Launchpad oder durch einen Doppelklick auf die App im Programme-Ordner.
    Da wir die App-Bundles derzeit nicht signieren können, erscheint beim ersten Start eventuell eine Warnung, dass QualCoder von einem nicht registrierten Entwickler stammt oder dass Apple die App nicht auf Schadsoftware prüfen konnte. Je nach Prozessor sehen Sie eine dieser Meldungen:
    
    Falls „Im Finder anzeigen“ oder „OK“ erscheint: Klicken Sie auf `OK`.
    
    Falls „Fertig“ oder „In den Papierkorb legen“ erscheint: Klicken Sie auf `Fertig`.
    
    Gehen Sie anschließend in die „Einstellungen“ und wählen Sie „Datenschutz & Sicherheit“. Scrollen Sie nach unten, bis Sie die Meldung sehen, dass das Starten von QualCoder verhindert wurde. Klicken Sie auf „Dennoch öffnen“. Möglicherweise müssen Sie Ihr Passwort eingeben und erneut auf „Öffnen“ klicken. Warten Sie nach der Passworteingabe, bis das Programm startet. Dies kann einige Sekunden dauern.
    
    Von nun an sollte QualCoder ohne Probleme starten.
    
    Sie können QualCoder auch ohne Administratorrechte installieren. Kopieren Sie dazu die Binärdateien in das Verzeichnis `/Users/mylogin/Applications`. Sie müssen das Quarantäne-Attribut des Anwendungsordners über das Terminal löschen:
    ```xattr -d com.apple.quarantine /Users/mylogin/Applications/QualCoder.app```
    
    **Installation aus dem Quellcode:**
    
    Laden Sie den Quellcode herunter. Lesen und befolgen Sie den Abschnitt für die macOS-Nutzung in der Datei `Readme.md`.
    Entfernen Sie die Backticks ( ` ), die die Befehle umschließen, da diese nur zur Darstellung dienen.

=== "Fedora Linux"

    Diese Anleitung lädt den aktuellen Quellcode direkt von GitHub herunter. Hinweis: Fedora verwendet `Wayland`, was möglicherweise nicht optimal mit der `Qt`-grafischen Benutzeroberfläche funktioniert. Es wird empfohlen, zusätzlich `Xwayland` zu installieren.
    
    Laden Sie den Quellcode herunter. Lesen und befolgen Sie den Abschnitt für die Fedora-Nutzung in der Datei `Readme.md`.
    Entfernen Sie die Backticks ( ` ), die die Befehle umschließen, da diese nur zur Darstellung dienen.
    **Es gibt ein bekanntes Problem beim Audio-/Video-Coding unter Linux Fedora – die Software stürzt ab.**
