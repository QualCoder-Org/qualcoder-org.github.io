Diese Anleitung beschreibt die Installation und Nutzung von QualCoder. Die Informationen beziehen sich auf die aktuellste Version von QualCoder.

## Was ist QualCoder?
QualCoder ist eine kostenlose Open-Source-Software für die qualitative Datenanalyse. 
Mit QualCoder können Sie Texte, Bilder, Audio- und Videodateien codieren sowie Journal-Notizen und Memos schreiben. Sie können Codes in einem baumartigen Kategorisierungsschema organisieren. Zudem lassen sich verschiedene Berichte erstellen. Die meisten Berichte können als `html`, Open Document Text (`ODT`), Plain Text oder `Excel`-Dateien exportiert werden.

Neuere Versionen enthalten optionale KI-gestützte Funktionen, die `GPT-4` oder andere Large Language Models nutzen, um Daten auf innovative Weise zu erkunden, zu analysieren und mit ihnen zu interagieren. Machen Sie sich mit neuen Daten vertraut, indem Sie [breite Themen oder Konzepte](5.1.-AI-chat-based-analysis#topic-analysis-chat/) in einem interaktiven Chat mit der KI untersuchen. Vertiefen Sie spezifische Aspekte mittels [KI-gestütztem Coding](4.2.-AI-Assisted-Coding/) und [Textanalyse](5.1.-AI-chat-based-analysis#text-analysis-chat/). Und wenn Sie an den Punkt kommen, an dem Sie Ihre Ergebnisse synthetisieren und konsolidieren müssen, können Sie diese in einem [Code-Chat](5.1.-AI-chat-based-analysis#code-analysis-chat/) mit der KI diskutieren.

!!! note ""
    QualCoder ist als clientbasierte Software konzipiert, die von einer Person zurzeit genutzt wird. (Ein zweiter Coder kann dasselbe Projekt auf demselben Computer nutzen, oder der Projektordner kann auf den Computer des zweiten Coders übertragen werden). QualCoder ist nicht für den gleichzeitigen Zugriff durch mehrere Personen ausgelegt. Es wird nicht empfohlen, das Projekt an einem Cloud-Speicherort abzulegen; dies kann die Datenbank beschädigen, falls die Verbindung während eines Aktualisierungsprozesses unterbrochen wird.
