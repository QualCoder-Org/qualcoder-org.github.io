# Einstellungen

Über den Einstellungsbildschirm können Sie verschiedene Funktionen anpassen, darunter die Sprache, den Coder-Namen, die Schriftart und Schriftgröße sowie die Optionen für die KI-Integration.

### Coder-Name

Mehrere Coder können denselben Text codieren und annotieren. Ihr aktueller Coder-Name wird oben im Einstellungsdialog angezeigt.

<img width="400" alt="grafik" src="https://github.com/user-attachments/assets/cd0aab19-5f2b-43de-b6f0-08ab171fbb91" />

Dieser Name dient dazu, Ihre Beiträge im Projekt zu identifizieren. Klicken Sie auf die Schaltfläche „Ändern“, um Ihre Identität anzupassen oder um andere Coder innerhalb des aktuellen Projekts zu erstellen, umzubenennen oder zusammenzuführen. [Weitere Informationen finden Sie hier.](2.4.-Working-in-a-Team.md/#important-considerations-regarding-coder-names)

### Allgemeine Einstellungen

Die Standardsprache von QualCoder ist Englisch. Als Alternativsprachen stehen Chinesisch, Französisch, Deutsch, Italienisch, Japanisch, Portugiesisch, Spanisch und Schwedisch zur Verfügung. Einige Sprachen wurden noch nicht vollständig auf die Genauigkeit der Übersetzung geprüft. Um die Sprache zu ändern, starten Sie QualCoder bitte neu.

Durch das Aktivieren des Kontrollkästchens „IDs anzeigen“ werden die numerischen IDs für Codes, Dateien, Cases und mehr eingeblendet. Diese IDs können beim Schreiben von `SQL`-Abfragen nützlich sein.

Es gibt zudem einige Optionen, um das allgemeine Erscheinungsbild von QualCoder anzupassen.

Für die Transkription von Audio- und Videodateien können das Zeitformat und der Klammertyp ausgewählt werden.

Sehr große Textdateien können den Coding-Modus in QualCoder verlangsamen. Daher gibt es im Bereich „Text codieren“ eine Rechtsklick-Option, um den Text in kleineren Abschnitten (Chunks) zu laden. Die Standardgröße beträgt 50.000 Zeichen, dies kann in den Einstellungen jedoch für langsamere Computer auf kleinere Werte reduziert werden.

### Backups

Die Projekt-Backups können so eingestellt werden, dass sie automatisch jede Stunde erstellt werden oder dass gar kein Backup erfolgt. QualCoder behält die aktuellsten Backups mit einem Datums- und Zeitstempel. Der Backup-Name lautet: `_projektname_BKP_jjjjmmtt_ss.qda_` (wobei `_ss_` die Uhrzeit im 24-Stunden-Format ist). Das bedeutet auch, dass innerhalb einer Stunde nicht mehrere Backups erstellt werden; ein neues Backup kann also kein anderes überschreiben, das in derselben Stunde erstellt wurde. Es wird nur eine begrenzte Anzahl an Backups aufbewahrt – **ältere Backups werden gelöscht**. In den Optionen können Sie wählen, ob zwischen 1 und 10 Backups gespeichert werden sollen.

Ich empfehle, vor größeren Änderungen – wie der Reorganisation von Codes und Kategorien, dem Zusammenführen von Projekten oder dem Zusammenführen von Codes – manuell ein Backup zu erstellen. QualCoder führt stündliche Backups durch (bis zu den letzten fünf), sofern dies in den Einstellungen aktiviert ist.

Benennen Sie Backups, die Sie dauerhaft behalten möchten, um oder verschieben Sie diese an einen anderen Ort. Wenn Backups aktiviert sind, gibt es die zusätzliche Option, auch Audio- und Videodateien zu sichern. Für die Vollständigkeit der Backups wird dies empfohlen; es kann jedoch das Starten der Software verlangsamen, wenn Sie viele große Dateien haben. Alternativ können Sie große Audio-/Videodateien verknüpfen, anstatt sie direkt in den Projektordner zu importieren. Das Standard-Projektverzeichnis ist der Ort, an dem die meisten Dateioperationen (z. B. für Exporte oder Importe) standardmäßig ansetzen.

In Ihrem Benutzerordner erstellt QualCoder einen Ordner `.qualcoder` mit einer Datei `QualCoder.log` zur Ereignisprotokollierung und einer Datei `config.ini`. Letztere enthält viele Einstellungen wie den Namen des aktuellen Coders, bevorzugte Schriftarten, die Sprache und ein bevorzugtes Arbeitsverzeichnis. Speichern Sie keine Projektdaten innerhalb des `.qualcoder`-Ordners. Die einzige Datei, die Sie hier ablegen sollten, ist eine `stopwords.txt`, falls Sie einen anderen Satz an Stoppwörtern für die WordCloud- und Wortzählfunktionen verwenden möchten. Dieser Ordner ist ausschließlich für die interne Nutzung der QualCoder-Software vorgesehen.
