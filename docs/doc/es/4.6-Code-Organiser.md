# Organizador de códigos

El organizador de códigos te permite cambiar la estructura del árbol de códigos.

Al abrirlo por primera vez, se te pedirá que hagas una copia de seguridad de tu proyecto antes de realizar y aplicar cualquier cambio, puesto que las modificaciones realizadas aquí no se pueden deshacer.

<img width="494" height="361" alt="image" src="https://github.com/user-attachments/assets/c429494b-a78f-4a1c-9b7b-f894a724843d" />

Utilizando el botón situado más a la izquierda, selecciona todas las ramas de código de tu proyecto, o bien, una rama de código en específico.

<img width="475" height="390" alt="image" src="https://github.com/user-attachments/assets/8d3cddb1-1fc1-44ec-ad45-23f54e5d40ce" />

Se mostrarán todas las subcategorías y códigos correspondientes a tu selección.

<img width="994" height="495" alt="image" src="https://github.com/user-attachments/assets/91505659-8e5d-43fe-9ee3-a139b25c3005" />

**Códigos:**

Haz clic derecho sobre un código para acceder a las opciones de: Eliminar de la categoría, Mover a otra categoría, Combinar con otro código, Ver/editar el memo, Renombrar, Ver los medios codificados, y Mostrar el memo debajo del nombre del código.

**Categorías:**

Hay un botón en la parte superior de la pantalla para añadir una nueva categoría. 

Haz clic derecho sobre una categoría para: combinar una categoría con otra, mover la categoría debajo de otra, eliminar la categoría de su posición bajo otra categoría principal, renombrar la categoría, ver y editar el memo, o simplemente mostrar el memo debajo del nombre de la categoría.

Mostrar el memo debajo del nombre de la categoría

Existe un botón para exportar la imagen como un archivo PNG.

El botón Aplicar (Apply) efectúa los cambios en la estructura del árbol de codificación. Si no se presiona, las modificaciones no se llevarán a cabo.
