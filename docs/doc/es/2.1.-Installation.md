# Instrucciones de Instalación y ejecución de QualCoder

## INSTALACIÓN

Para las instalaciones manuales (mediante la línea de comandos o terminal), se hace referencia a una versión específica de lanzamiento (por ejemplo, la versión 3.8). Te recomendamos consultar la [página de lanzamientos (releases)](https://github.com/ccbogel/QualCoder/releases) para obtener la versión más reciente, ya que podría existir una actualización posterior. 

### Requisitos Previos
VLC es opcional; no obstante, su instalación es importante si requieres realizar codificación de archivos de audio o video. En ciertos sistemas operativos (por ejemplo, Fedora), es posible que _QualCoder_ presente dificultades para localizar la ruta de instalación de VLC. De manera opcional, se recomienda instalar FFmpeg para habilitar la visualización de formas de onda (waveform); para mayor información sobre este procedimiento, consulta: https://phoenixnap.com/kb/ffmpeg-windows.

### MacOS

**Instalación desde paquetes de aplicación (App bundles):**

Asegúrate de haber instalado todas las actualizaciones disponibles de macOS antes de iniciar el proceso de instalación. Dirígete a la aplicación Ajustes del Sistema en tu Mac. En la barra lateral, haz clic en General (es posible que debas desplazarte hacia abajo) y, posteriormente, selecciona Actualización de software.

Instalación de _QualCoder_: Haz doble clic en el archivo .dmg descargado. Posteriormente, arrastra el icono de _QualCoder_ hacia el acceso directo de la carpeta de Aplicaciones.

Inicia _QualCoder_ desde el Launchpad o haciendo doble clic en la aplicación dentro de tu carpeta de Aplicaciones.
Actualmente, no disponemos de la capacidad para firmar los paquetes de aplicación (app bundles). Por lo tanto, al intentar ejecutar el programa por primera vez, es posible que el sistema muestre una advertencia emergente indicando que _QualCoder_ proviene de un desarrollador no registrado, o bien el mensaje: "_QualCoder_ no se puede abrir porque Apple no puede realizar un análisis en busca de software malicioso". Dependiendo del tipo de procesador de tu equipo, podrías visualizar una de las siguientes opciones:

Si visualizas las opciones "Mostrar en el Finder" o "Aceptar", haz clic en Aceptar.

Si visualizas las opciones "Listo" o "Trasladar a la papelera", haz clic en Listo.

Posteriormente, dirígete a Ajustes y selecciona la sección Privacidad y seguridad. Desplázate hacia abajo hasta localizar un mensaje indicando que se impidió la ejecución de _QualCoder_. Haz clic en el botón "Abrir de todos modos". Es posible que el sistema te solicite ingresar tu contraseña y que debas confirmar la acción haciendo clic nuevamente en "Abrir". Tras validar tu identidad, espera a que el programa se inicie; este proceso puede demorar varios segundos.

A partir de este momento, _QualCoder_ debería ejecutarse sin inconvenientes.
