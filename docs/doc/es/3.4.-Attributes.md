# Atributos

Los atributos vinculan variables contextuales a sus archivos, casos o diarios. Desde el menú Administrar > Administrar atributos , puede crear, editar, eliminar o añadir memos para enriquecer la interpretación de sus datos (Véase Imagen 1).

Puede acceder al administrador de atributos de dos maneras distintas:

**1. Atajo de teclado:** Presionando simultáneamente las teclas **ALT + A**.

**2. Menú _Administrar_:** Seleccionando la opción "Administrar Atributos" dentro de la pestaña desplegable.

>> Puede hacer clic derecho sobre un atributo numérico para convertirlo a tipo Carácter (Character).

Las únicas opciones para los atributos son numérico y carácter. No existen opciones para tipos booleanos, fecha, hora o tipos numéricos específicos como enteros (Integers). Sin embargo, si está ejecutando consultas SQL, es posible que necesite realizar una conversión (cast) del atributo numérico a Entero (Integer) o Real: cast(my_var as integer).

Los atributos pueden añadirse en los administradores de archivos, casos o diarios haciendo clic en el botón (x).

Es posible importar atributos desde una encuesta y también desde un archivo de proyecto REFI-QDA. Tenga en cuenta que no se garantiza que la importación de proyectos REFI-QDA se realice con éxito en todos los casos.

QualCoder no tiene como fin realizar análisis estadísticos de los atributos. Para ello, es preferible utilizar software estadístico especializado como R (https://www.r-project.org/).

También puede eliminar atributos. Aunque es posible eliminar atributos de referencias, recomendamos no hacerlo a menos que esté seguro de que no necesitará ninguna de sus referencias. Estos atributos tienen el prefijo 'Ref_'.
