# Desarrollo de QualCoder

## Estructuras de los datos

La carpeta de datos (proyecto) **.qda** contiene carpetas para los documentos, imágenes, audio y video importados. También contiene la base de datos SQLite, llamada **data.qda**, donde se almacenan los datos codificados.

QualCoder crea los archivos QualCoder.log y QualCoder_settings.txt en una carpeta llamada .qualcoder dentro de su directorio personal. El archivo de configuración contiene el nombre del codificador actual, el directorio de trabajo predeterminado y la elección de la fuente. El archivo de registro (log) registra los errores del programa y algunas acciones del usuario.

Cuando la IA está habilitada: se crea una carpeta llamada **ai_data** que contiene una base de datos SQLite con el historial del chat y un vectorstore (almacén de vectores). Un vectorstore es una representación numérica de palabras, frases y oraciones utilizada por el modelo de lenguaje de la IA. El vectorstore emplea FAISS, que se utiliza para las búsquedas de similitud de la IA.

QualCoder está escrito en Python 3 y utiliza Qt6 para la interfaz gráfica.


## Uso

Actualmente, QualCoder se desarrolla en Windows 11.

La carpeta del proyecto .qda

La carpeta de datos qda contiene carpetas para los documentos, imágenes, audio y video importados. También contiene la base de datos SQLite, llamada data.qda, para almacenar los datos de la codificación, y la carpeta ai_data que contiene la base de datos SQLite para los requerimientos de la IA.

La carpeta de configuración .qualcoder

QualCoder crea los archivos QualCoder.log y QualCoder_settings.txt dentro de una carpeta llamada .qualcoder en su directorio personal (home). El archivo de configuración contiene el nombre del codificador actual, el directorio de trabajo predeterminado, la fuente, los idiomas y otras opciones. El archivo de registro (log) registra los errores del programa y algunas acciones del usuario; es posible que existan varios archivos de registro rotativos. Usted no debe colocar ningún archivo dentro de esta carpeta, ya que se utiliza como área de almacenamiento de la configuración de trabajo del programa. Sin embargo, puede colocar un archivo llamado stopwords.txt en esta carpeta. Este archivo de palabras vacías (stopwords) anulará la lista integrada de palabras vacías en inglés.

Se recomienda una resolución de pantalla mínima de 1024x600 píxeles.

**Sistemas operativos compatibles:**

QualCoder ha sido utilizado en Ubuntu 24, Fedora, Linux Mint, Arch Linux, Windows 10, Windows 11 y macOS. El desarrollo del software se realiza en Windows 11. En Fedora, pueden presentarse inconvenientes con el uso de audio y video; esto es un problema que aún no se ha resuelto.

En ocasiones, para diversas distribuciones de Linux, pueden requerirse instrucciones de instalación distintas. No contamos con la capacidad para identificar y solucionar todos estos casos, por lo que dependemos de los comentarios o de las instrucciones mejoradas de los usuarios finales para resolver estos inconvenientes.
