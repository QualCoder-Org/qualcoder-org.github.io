# Gráfico (Grafo)

En este informe se visualiza un gráfico (Grafo/red) de tus categorías y códigos (Ver imagen 1 y tabla 1). Cuentas con opciones para limitar los elementos que se muestran, como seleccionar "Todos" o elegir únicamente "Categorías seleccionadas". Al hacer clic en el área de edición, puedes utilizar las teclas + o - para acercar o alejar la vista.

Cada código o categoría puede desplazarse dentro del área de trabajo; para hacerlo, haz clic cerca del borde del recuadro y arrástralo a la posición deseada. Si haces clic directamente sobre el texto, podrás modificar temporalmente el nombre del código o la categoría. Al hacer clic derecho sobre un código, se desplegará un menú con opciones como mostrar el memo o visualizar todo el texto de los casos o archivos que han sido codificados con dicho código.

Al hacer clic derecho sobre una línea, puedes modificar su grosor, cambiar el estilo a línea punteada o ajustar su color, lo cual resulta útil para enfatizar relaciones específicas. Si haces clic derecho en un espacio vacío, podrás añadir cuadros de texto libre, de archivos o de casos, así como insertar líneas adicionales entre los distintos componentes.

El sistema permite mostrar los segmentos codificados de texto, imágenes y fragmentos de audio o video (A/V). Asimismo, puedes visualizar los memos asociados a estos segmentos en cuadros de texto independientes. Cada uno de estos elementos cuenta con etiquetas informativas (_tooltips_) que proporcionan detalles adicionales. Toma en cuenta que no es posible mostrar el mismo segmento codificado más de una vez dentro del mismo gráfico.

Los gráficos pueden guardarse, cargarse y eliminarse según lo requieras. Además, tienes la opción de exportarlos como archivos en formato de imagen **PNG**.
