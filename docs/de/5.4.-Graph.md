# Graph

Es wird ein Graph der Kategorien und Codes angezeigt. Es gibt Optionen, um die angezeigten Codes und Kategorien einzuschränken – zum Beispiel „Alle“ oder „Ausgewählte Kategorien“. Ein Klick auf den Bildbereich ermöglicht das Zoomen mit `+` oder `-`.

Jeder Code oder jede Kategorie kann verschoben werden, indem man in der Nähe des Randes des Kastens klickt und diesen zieht. Ein Klick auf den Text ermöglicht es, die Bezeichnung des Codes oder der Kategorie temporär zu ändern. Ein Rechtsklick auf einen Code öffnet ein Menü mit Optionen, wie zum Beispiel das Anzeigen des Memos oder das Anzeigen aller Case- oder Dateitexte, die mit dem ausgewählten Code codiert wurden.

Ein Rechtsklick auf eine Verbindungslinie ermöglicht es, die Dicke zu ändern, den Stil auf eine gepunktete Linie umzustellen oder die Linienfarbe zu ändern. Dies kann zur Hervorhebung genutzt werden. Ein Rechtsklick auf eine leere Fläche erlaubt das Hinzufügen von freien Textfeldern, Dateitextfeldern und Case-Textfeldern sowie das Einfügen zusätzlicher Linien zwischen den Komponenten.

Codierter Text, codierte Bilder und codierte A/V-Segmente können angezeigt werden. Memos von codiertem Text, codierten Bildern und codierten A/V-Segmenten können in Textfeldern dargestellt werden. Jedes dieser Elemente verfügt über Tooltips, die weitere Informationen liefern. Ein und derselbe codierte Abschnitt kann nicht mehrfach im Graph angezeigt werden.

Graphen können gespeichert, geladen und gelöscht werden. Zudem können Graphen als `PNG`-Dateien exportiert werden.
