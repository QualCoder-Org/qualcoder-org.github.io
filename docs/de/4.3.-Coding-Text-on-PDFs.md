# PDFs codieren

PDFs können ähnlich wie das Originaldokument angezeigt werden, und der Text im PDF kann codiert werden. Die in QualCoder angezeigten PDFs sind keine exakten Kopien, sondern werden mit einigen Einschränkungen (siehe unten) dargestellt. PDF-Dokumente benötigen eine längere Ladezeit; größere PDFs laden entsprechend langsamer. Bei kleinen PDFs sind einige Sekunden, bei großen PDFs mehrere Minuten Wartezeit möglich. Bei sehr großen PDFs wird empfohlen, nur den Plain Text über die Option „Text codieren“ zu bearbeiten. Zudem ist es ratsam, PDFs mit einer Länge von maximal etwa 20 Seiten zu verwenden.

Der Bildschirm ist in drei anpassbare Bereiche unterteilt. Der linke Bereich enthält die Dateiliste und den Code-Baum. Der mittlere Bereich zeigt die PDF-Darstellung. Der rechte Bereich zeigt den aus dem PDF extrahierten Plain Text.

## Schriftgröße und Zoom

Die Originalschriften des Textes werden nicht verwendet, da deren Extraktion und Anwendung aus dem PDF zu komplex ist. Stattdessen wird in den angezeigten PDF-Textfeldern die Standard-Schriftart von QualCoder genutzt. Ein Regler für die Schriftgröße ermöglicht es, die Schrift zu verkleinern, um Textüberlappungen zu vermeiden. Standardmäßig ist dieser auf -2 eingestellt; ein Wert von -3 kann ebenfalls vorteilhaft sein. Das PDF kann über Schaltflächen oder Tastenkombinationen gezoomt werden: `Strg` + `-` und `Shift` + `+`.

## Übereinstimmung des Plain Text

Der Plain Text, der beim Import eines PDFs über „Dateien verwalten“ geladen wird, muss exakt mit dem Plain Text übereinstimmen, der beim Anzeigen des PDFs erneut geladen wird. Das bedeutet, dass PDFs, die vor Version 3.4 geladen wurden, den Text nicht korrekt zuordnen. Wenn die Textlängen nicht übereinstimmen, wird das Codieren über die Auswahl von Textfeldern deaktiviert. Ebenso wird diese Funktion deaktiviert, wenn der Plain Text in irgendeiner Weise bearbeitet wurde und die Längen nicht mehr übereinstimmen. In diesem Fall kann auch die Darstellung der Coding-Markierungen auf der PDF-Seite fehlerhaft sein.

## Zusatzfunktionen 

Am oberen Fensterrand befinden sich mehrere Schaltflächen. Eine davon exportiert ein Bild der PDF-Seite. Die Lupen-Schaltfläche öffnet ein Fenster, das die auf der Seite verwendeten PDF-Elemente anzeigt.

Zudem gibt es Kontrollkästchen zum Ein- und Ausblenden von PDF-Elementen (Text, Bilder). Ein Kontrollkästchen stellt allen Text in Schwarz dar, was hilfreich ist, wenn hellfarbiger Text auf hellem Hintergrund erscheint.

## Rechter Bereich

Der rechte Bereich zeigt den Plain Text in derselben Weise an wie im Modus „Text codieren“. Hier können codierte Texte markiert, entmarkiert, mit Memos versehen, als wichtig markiert oder annotiert werden.

## Mittlerer Bereich

Zum Codieren wird die Maus mit einem Linksklick gedrückt und gezogen, um PDF-Textfelder auszuwählen. Dies kann je nach Größe und Position der Felder schwierig sein. Anschließend wird ein Code angeklickt, um die Auswahl diesem zuzuordnen. Weitere Anpassungen können im rechten Plain-Text-Bereich vorgenommen werden: mit `Shift` + `Links`/`Rechts` wird der codierte Abschnitt erweitert, mit `Alt` + `Links`/`Rechts` wird er verkleinert.

Ein Rechtsklick auf ein codiertes Segment öffnet das Kontextmenü, über das Memos hinzugefügt, die Markierung als „wichtig“ gesetzt oder das Segment entmarkiert werden kann.

### Tastenkombinationen

`H`: Obere Bildschirmbereiche aus- oder einblenden.

`Strg F`: Springt zum Textsuchfeld.

`Strg Z`: Macht das letzte Entmarkieren eines Codes rückgängig.

`Strg 1`: Wechselt zur nächsten Datei.

`Strg 2`: Wechselt zur zuletzt codierten Datei.

`Strg 4`: Dateimemo anzeigen und bearbeiten.

`Strg 5`: Dateiliste basierend auf Attributen abrufen.

`Strg 9`: Wichtige codierte Segmente ein- oder ausblenden.

`Strg 0`: Hilfeseite im Browser öffnen.

#### Im PDF-Anzeigebereich:

Durch Klicken auf das PDF und Drücken der Tasten `+` oder `-` kann gezoomt werden.
`Q`: Auswahl schnell mit dem letzten Code markieren.
`R`: Auswahl schnell mit einem der kürzlich verwendeten Codes markieren.

#### Im Plain-Text-Bereich:

`A`: Annotieren (für aktuelle Auswahl – nur im Textbearbeitungsmodus).

`Q`: Quick-Markierung mit Code (für aktuelle Auswahl).

`I`: Als wichtig markieren.

`M`: Memo zum Code hinzufügen (an der angeklickten Position).

`O`: Durch überlappende Codes an der angeklickten Position wechseln.

`R`: Kontextmenü für kürzlich verwendete Codes zum Markieren von Text öffnen.

`U`: An der ausgewählten Stelle entmarkieren.

`V`: „In-vivo“-Code an den ausgewählten Text zuweisen.
