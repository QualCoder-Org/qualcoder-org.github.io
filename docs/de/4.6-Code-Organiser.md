# Code-Verwaltung

Die Code-Verwaltung (Code Organizer) ermöglicht die Änderung der Struktur des Coding-Baums.

Beim ersten Öffnen erscheint die Aufforderung, ein Backup des Projekts zu erstellen, bevor Änderungen vorgenommen oder angewendet werden. Änderungen in diesem Bereich können nicht rückgängig gemacht werden.

![image](https://github.com/user-attachments/assets/ae81b366-54e8-457f-9afe-abb7f76d2a41)

Über die ganz linke Schaltfläche werden entweder alle Code-Zweige des Projekts oder nur ein spezifischer Code-Zweig ausgewählt.

![image](https://github.com/user-attachments/assets/57b82319-1145-4872-a6c7-63cbc807b370)

Alle Unterkategorien und Codes der Auswahl werden anschließend angezeigt.

![image](https://github.com/user-attachments/assets/303ddd26-605c-4963-9db4-ba7c50c40250)

**Codes:**

Ein Rechtsklick auf einen Code bietet folgende Optionen: Entfernen aus der Kategorie, Verschieben in eine andere Kategorie, Zusammenführen mit einem anderen Code, Memo anzeigen/bearbeiten, Umbenennen, codierte Medien anzeigen sowie das Memo unter dem Code-Namen einblenden.

**Kategorien:**

Am oberen Bildschirmrand befindet sich eine Schaltfläche zum Hinzufügen einer neuen Kategorie.

Ein Rechtsklick auf eine Kategorie ermöglicht: das Zusammenführen einer Kategorie mit einer anderen, das Verschieben einer Kategorie unter eine andere, das Entfernen einer Kategorie aus einer übergeordneten Kategorie, das Umbenennen einer Kategorie, das Anzeigen und Bearbeiten des Memos oder das Einblenden des Memos unter dem Kategorienamen.

Einblenden des Memos unter dem Kategorienamen:

![image](https://github.com/user-attachments/assets/92fee31b-1b70-4bdf-bb15-51b0485e4739)

Über eine Schaltfläche kann die Ansicht als `PNG`-Datei exportiert werden.

Die Schaltfläche „Anwenden“ (Apply) übernimmt die Änderungen an der Struktur des Coding-Baums. Wenn diese nicht gedrückt wird, werden die Änderungen nicht übernommen.
