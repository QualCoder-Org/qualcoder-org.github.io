# Audio und Video codieren

Beim Codieren von Videos werden zwei Bildschirme angezeigt, wie unten abgebildet. Ein Bildschirm zeigt die Videowiedergabe. Der andere Hauptbildschirm enthält die Steuerelemente sowie eine Schaltfläche zum Starten und Beenden eines codierten Segments. Das Steuerungsfenster zeigt die Codes im unteren linken Bereich und den Text der `.transcribed`-Datei im unteren rechten Bereich an. Beim Codieren von Audio wird nur der Hauptbildschirm angezeigt.

Der Transkriptionstext kann in diesem Fenster ebenfalls codiert und annotiert werden. Ein Rechtsklick auf einen Zeitstempel in der Transkription öffnet ein Menü, um direkt zu diesem Abschnitt des Videos zu springen. Falls ein Video mehrere Audiospuren besitzt, kann die Audiospur gewechselt werden. Durch Aktivieren des Kontrollkästchens „Scrollen“ bewegt sich das Transkript synchron zum Video basierend auf den erkannten Zeitstempeln; zum Codieren des Transkripts muss diese Option jedoch deaktiviert werden.

## Tastenkombinationen

* `Strg R`: 5 Sekunden zurückspulen
* `Alt R` oder `Alt Minus`: 30 Sekunden zurückspulen
* `Alt F` oder `Alt Plus`: 30 Sekunden vorspulen
* `Strg P` oder `Strg D`: Audio/Video Start/Stop (Play/Pause). Beim Fortsetzen wird automatisch eine Sekunde zurückgespult.
* `Strg S`: Erstellung eines A/V-Segments starten/beenden.
* `Strg Shift <`: Wiedergabegeschwindigkeit auf bis zu 0,1 verringern.
* `Strg Shift >`: Wiedergabegeschwindigkeit bis zu 2-fach erhöhen.
* `Strg Z`: Letztes entmarkiertes Coding wiederherstellen.

(Diese Optionen sind auch über die Symbole in der Symbolleiste verfügbar.)

Wenn der Textbereich fokussiert ist:

* `A`: Markierten Text annotieren.
* `L`: „Codes anzeigen wie“ (Show codes like), um eine Kurzliste im Code-Baum anzuzeigen.
* `M`: Memo zu einem Code hinzufügen (Klick auf codiertes Segment erforderlich).
* `O`: Durch überlappende Codes an der aktuellen Stelle wechseln (Klick auf überlappendes Segment erforderlich – durch Oberlinie gekennzeichnet).
* `Q`: Quick-Code mit dem zuletzt verwendeten Code (Textauswahl erforderlich).
* `R`: Kürzlich verwendete Codes auswählen (Textauswahl erforderlich).

## Audio- und Video-Segmente

Sobald ein Segment über die Schaltfläche „Segment starten/Segment beenden“ erstellt wurde, kann dieses einem Code zugewiesen werden, indem ein Rechtsklick auf den entsprechenden Code erfolgt.

Die Coding-Markierungen für die codierten Segmente werden im oberen Bereich angezeigt. Das Bewegen des Mauszeigers über eine Markierung zeigt den Code-Namen, das Zeitsegment und etwaige Memos an. Ein Rechtsklick auf eine Markierung öffnet ein Menü zum Bearbeiten des Memos, zum Markieren als „Wichtig“ oder zum Löschen des Segments. Die Markierungen werden auf verschiedenen Zeilen dargestellt, um Überlappungen zu vermeiden.

Der Transkriptionstext kann in diesem Dialog ebenfalls codiert und annotiert werden. Dies ist jedoch nicht möglich, wenn das Kontrollkästchen „Transkript scrollen“ aktiviert ist. Bei aktivierter Option scrollt das Transkript synchron zum Video anhand der Zeitstempel.

## Wichtige codierte Texte und Segmente

Wichtige codierte Texte, wie etwa Exemplare für einen Bericht, können über das Rechtsklick-Menü als „Wichtig“ markiert werden. Ebenso können Code-Segmente als wichtig markiert werden. Am oberen Bildschirmrand befindet sich eine Schaltfläche mit einem Stern-Symbol, die zwischen der Anzeige aller codierten Texte/Segmente und nur der als wichtig markierten Texte/Segmente umschaltet.

**Text an codierte Segmente verknüpfen**

Text auswählen (dieser kann bereits codiert oder vorzugsweise uncodiert sein) und diesem einem Segment zuweisen. Zuerst den Text markieren, dann per Rechtsklick auf die Segment-Markierung das Menü öffnen und „Text an Segment verknüpfen“ wählen.

## Code-Positionen im Textbereich anpassen

Im Textbereich auf einen Code klicken (Hinweis: der Code darf an dieser Position nicht mit einem anderen Code überlappen). Folgende Tastenkombinationen erweitern oder verkleinern das codierte Textsegment:

* `Shift` + `Linkspfeil`: Codierten Text nach links erweitern.
* `Shift` + `Rechtspfeil`: Codierten Text nach rechts erweit erweitern.
* `Alt` + `Linkspfeil`: Codierten Text von rechts nach links verkleinern.
* `Alt` + `Rechtspfeil`: Codierten Text von links nach rechts verkleinern.

Zudem ist es möglich, per Rechtsklick auf einen Code die Start- oder Endposition um eine bestimmte Anzahl an Zeichen zu ändern.

## Das Videofenster

Im Videofenster lassen sich per Rechtsklick die Fenstergröße in Pixeln anpassen. Zudem kann ein Screenshot exportiert werden. Dieser wird als `Frame_jjjjmmtt_ss_mm_ss.jpg` gespeichert.

## Komfort-Schaltflächen unter der Dateiliste

* Nächste Datei laden.
* Datei mit dem aktuellsten Coding laden.
* Dateimemo der ausgewählten Datei öffnen.
* Filterfunktion nutzen, um nur Dateien anzuzeigen, die bestimmte Kriterien erfüllen (z. B. Datei-Attribute oder Zuordnung zu bestimmten Cases).

## Audio- und Videoprobleme unter bestimmten Betriebssystemen

Einige Betriebssysteme funktionieren nicht optimal mit Audio und Video in QualCoder (z. B. Fedora Linux).

Es ist eine aktuelle 64-Bit-Version von `VLC` zu installieren, da QualCoder diese für die Audio- und Videowiedergabe nutzt.

# Kategorien und Codes

Kategorien dienen der Organisation von Codes. Sie sind hierarchisch in einer Baumstruktur organisiert. Codes können in Kategorien verschoben und Kategorien in übergeordnete Kategorien verschoben werden. Kategorien und Codes können aus ihrer aktuellen Position herausgelöst werden. Codes und Kategorien können zusammengeführt werden, indem ein Code auf einen anderen Code oder eine Kategorie auf eine andere Kategorie gezogen wird. Kategorien und Codes können mit Memos versehen werden. Per Rechtsklick können Kategorien oder Codes umbenannt werden. Ein Rechtsklick auf einen Code ermöglicht die Änderung der Farbe.

Um die Anzahl der im Code-Baum angezeigten Codes zu verringern, im Coding-Fenster (Text, Audio/Video oder Bild) per Rechtsklick auf den Code-Baum die Option „Codes anzeigen wie“ (Show codes like) wählen. Dann einen Text in das Feld eingeben. Es werden nur noch Codes angezeigt, die diesen Text enthalten. Um wieder alle Codes anzuzeigen, das Feld leeren und mit `OK` bestätigen.

Im Code-Baum auf der linken Seite öffnet ein Rechtsklick auf einen Code ein Kontextmenü mit verschiedenen Optionen.
Eine Option ist „Codierte Dateien anzeigen“ (Show coded files) – dies zeigt alle Stellen an, an denen mit dem ausgewählten Code codiert wurde. Ein Klick auf einen bestimmten codierten Abschnitt öffnet diesen im Originalkontext für weitere Einblicke.

In der Ansicht „Codierte Dateien anzeigen“ ermöglicht ein Rechtsklick auf den farbigen Kopfbereich weitere Optionen, wie das Hinzufügen weiterer Codes, das Entfernen des Codes oder den Export des gesamten Satzes codierter Segmente in eine Textdatei.

## Barrierefreiheit und Code-Farben

Im Menü „Coding“ gibt es die Option **Farbschema** (Colour scheme).

Hier kann eine Reihe von Farben auf eine Auswahl von Codes angewendet werden, anstatt sich auf zufällig generierte Farben oder die Einzeländerung jedes Codes zu verlassen. In diesem Bereich kann zudem eingesehen werden, wie die Code-Farben für Personen mit verschiedenen Formen von Farbenblindheit erscheinen könnten: Rot-Schwäche, Rot-Blindheit, Grün-Schwäche oder Grün-Blindheit.
