# Graphe

Un graphe présentant les catégories et les codes s'affiche. Des options permettent de limiter les codes et les catégories affichés, par exemple « Toutes » ou « Catégories sélectionnées ». Cliquez sur la zone de l'image, puis utilisez les boutons « + » ou « – » pour zoomer ou dézoomer.

Chaque code ou catégorie peut être déplacé en cliquant près du bord de la case et en la faisant glisser. Cliquer sur le texte vous permet de modifier temporairement la libellé du code ou de la catégorie. Un clic droit sur un code affiche un menu d'options telles que l'affichage de la note, ou l'affichage de tout le texte du dossier ou du cas qui a été codé avec le code sélectionné.

Un clic droit sur une ligne vous permet de modifier son épaisseur, de passer à un style de ligne pointillée ou de changer la couleur de la ligne. Cela peut être utile pour mettre en évidence certains éléments. Un clic droit sur un espace vide vous permet d’ajouter des zones de texte libre, des zones de texte de dossier et des zones de texte de dossier, ainsi que d’insérer des lignes supplémentaires entre les composants.

Le texte codé, les images codées et les segments audiovisuels codés peuvent être affichés. Les notes relatives au texte codé, aux images codées et aux segments audiovisuels codés peuvent être affichées dans des zones de texte. Chacune d'entre elles comporte des info-bulles fournissant des informations supplémentaires. Vous ne pouvez pas afficher plusieurs fois le même segment codé dans le graphe.

Les graphes peuvent être enregistrés, chargés et supprimés. Les graphes peuvent être exportés au format PNG.
