# Gérer les cas

Les dossiers permettent de regrouper divers fichiers liés à une situation ou à une personne en particulier. Ils peuvent comporter des attributs tels que l'âge et le sexe des participants. Vous pouvez ajouter une note à chaque dossier et renommer un dossier en double-cliquant sur son nom. Tous les projets ne nécessitent pas de dossiers ; par exemple, un projet d'entretiens individuels serait chargé sous forme de transcriptions textuelles individuelles, et ces fichiers peuvent se voir attribuer des attributs dans « Gérer les fichiers », tels que le rôle d'une personne, son âge, etc.

Un tableau répertorie les dossiers et leurs attributs. Le nombre de fichiers associés à un dossier est indiqué. Cliquez sur la cellule « Fichiers » du tableau pour ajouter ou modifier les fichiers associés à ce dossier. Cela affiche la boîte de dialogue du gestionnaire de dossiers, décrite ci-dessous.

Lorsqu’une enquête est importée, chaque ligne de l’enquête est attribuée à un cas. Le nom du cas correspond à l’identifiant unique figurant dans la première colonne du fichier CSV ou de la feuille de calcul Excel. L’importation d’enquêtes volumineuses (par exemple, 1 000 lignes ou plus) peut prendre beaucoup de temps, car la création des cas nécessite un certain temps.

Les boutons disponibles ici :

* Créer un nouveau cas
* Ajouter un attribut de cas
* Importer des cas à partir d'un fichier CSV ou XLSX
* Ouvrir le gestionnaire de fichiers de cas
* Exporter les attributs de cas vers un fichier XLSX
* Supprimer un cas
* Bouton d'aide

## Masquer et afficher les lignes et colonnes

Masquez les colonnes en faisant un clic droit sur la ligne d'en-tête du tableau pour obtenir des options permettant de masquer cette colonne ou les colonnes commençant par un texte spécifique. Ou pour afficher les colonnes commençant par une valeur textuelle. Masquez les lignes via le menu contextuel (clic droit) en sélectionnant une valeur textuelle qui est égale ou « similaire » à un texte choisi. Faites un clic droit pour trier par nom de cas ou par valeur d'attribut, par ordre croissant ou décroissant.

!!! example "Exemple pratique"
    Ajoutez les trois étudiants (ou cas) en cliquant sur l'icône Ajouter un cas. Nommez chaque étudiant : ID1, ID2 et ID3.

## Gestionnaire de fichiers de cas

Ajoutez maintenant le dossier de chaque étudiant à son dossier personnel. Cliquez sur la cellule de la colonne « Fichiers » correspondant au dossier ID3, par exemple. Cela ouvre la boîte de dialogue du gestionnaire de dossiers. Vous pouvez également cliquer sur l'icône « Documents », ce qui ouvrira également le gestionnaire de dossiers.

![Case File Manager](/images/case-file-manager.png)

Dans le _gestionnaire de fichiers de cas_, cliquez sur un cas, par exemple ID3, puis cliquez sur le bouton « Ajouter les fichiers sélectionnés au cas ». Sélectionnez un ou plusieurs fichiers dans la liste des fichiers. Ajoutez les fichiers au cas. Par  exemple, ajoutez le fichier miguel-henriques.jpg au cas ID3. Vous pouvez supprimer des fichiers et afficher les fichiers associés au cas. Vous verrez que le texte du fichier sera souligné en rouge, ce qui indique que ce texte est associé à ce cas.

Vous pouvez également sélectionner un dossier, surligner manuellement des portions de texte dans un document plus volumineux, puis cliquer avec le bouton droit pour marquer – ou attribuer – le texte au dossier sélectionné. 


## Importer des cas et des attributs de cas

Vous pouvez importer des attributs depuis un fichier CSV ou XLSX en utilisant ce bouton :
<img width="37" height="38" alt="image" src="https://github.com/user-attachments/assets/0b9d8a34-33ff-4591-ae6d-c6f43bfb9625" />

La première ligne doit contenir les en-têtes des attributs. La première colonne doit contenir les noms de dossiers correspondants (dans cet exemple : ID1, ID2, ID3). Notez que la distinction entre majuscules et minuscules est prise en compte. Ouvrez le fichier cases.csv dans le dossier « Examples » pour voir comment il est structuré.

Pour cet exemple, supprimez tous les cas affichés. Ensuite, importez les attributs du fichier cases.csv via la boîte de dialogue Importer les attributs. Vous devrez à nouveau associer les fichiers et le texte des fichiers à chaque cas.
