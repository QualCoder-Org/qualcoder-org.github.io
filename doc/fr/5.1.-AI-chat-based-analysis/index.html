    Analyse fondé sur le chat IA

    L'onglet Chat IA de la fenêtre principale vous permet de discuter de certains aspects de votre analyse avec l'IA. QualCoder propose quatre types de chats différents :

    Note : L'IA doit être totalement chargée et prête pour de nouvelles requêtes (« IA : prêt » dans la barre d'état) avant de pouvoir commencer un chat.

    Sur le côté gauche de l'onglet Chat IA, vous verrez une liste de tous les chats précédents de votre projet. Double-cliquez sur un chat (ou appuyez sur F2) pour modifier le titre afin de mieux organiser vos échanges.

    Chat d'analyse thématique

    • Donne un aperçu rapide d'un thème, d'un concept ou d'un phénomène dans votre corpus empirique, sans avoir à coder vos données au préalable.
    • Utilisez-le comme première exploration pour vous familiariser avec les données.
    • Basé sur la même technologie de recherche qui alimente le Codage assisté par IA.

    Dans l'onglet Chat IA, cliquez sur « Nouveau > Nouveau chat d'analyse thématique », et la fenêtre suivante apparaîtra :

    1) Saisissez un thème que vous souhaitez explorer et, éventuellement, une description. Notez que vous n'avez pas besoin d'entrer des mots-clés de recherche exacts ici. Vous pouvez explorer des concepts beaucoup plus abstraits (par exemple, « l'évolution des rôles de genre »).

    2) Sélectionnez une instruction analytique : - « Résumé du thème » vous donnera un bref résumé des résultats, généralement sous forme de puces. - « Analyser l'inattendu » tentera d'extraire de nouvelles informations surprenantes et inattendues de vos données en utilisant les vastes connaissances de bon sens intégrées dans les grands modèles de langage comme toile de fond pour la comparaison. - « Analyser les différences » recherchera spécifiquement les différences entre les cas, les documents, etc., dans les données.

    (En option avancée, vous pouvez également définir vos propres instructions analytiques en cliquant sur le bouton « Modifier », qui ouvre la Bibliothèque de instructions IA.)

    3) En troisième étape, vous pouvez ajouter des filtres (documents, cas, attributs) pour limiter les données incluses dans l'analyse.

    Après avoir cliqué sur « OK », la recherche initiale de données pertinentes dans la mémoire locale de l'IA est effectuée. Après un certain temps, l'analyse commencera à s'afficher à l'écran.

    L'IA ajoutera des références aux sources originales sous forme de liens hypertextes (bleus, soulignés). Passez votre souris sur la référence pour voir le fragment de données sous-jacent. Cliquer sur le lien hypertexte ouvrira le fragment dans l'espace de codage de texte afin que vous puissiez le voir dans son contexte.

    N'oubliez pas qu'il s'agit d'un chat : vous pouvez poser des questions de suivi ou discuter de l'interprétation avec l'IA. Cependant, il n'est actuellement pas possible d'ajouter un nouveau thème ou de nouvelles données à la conversation. Si vous souhaitez approfondir un sous-thème ou explorer un autre aspect, commencez un nouveau chat.

    Chat d'analyse de texte

    • Permet d'analyser des passages sélectionnés de documents empiriques avec l'IA.
    • Fournit un soutien ou des idées supplémentaires pendant la phase d'interprétation détaillée du texte.

    Cette analyse basculera d'abord vers l'espace de codage de texte (Menu : Codage > Coder le texte). Là, vous pouvez ouvrir un document, sélectionner le texte que vous souhaitez analyser, faire un clic droit dessus et choisir « Analyse de texte par IA » dans le menu contextuel (sera grisé si l'IA n'est pas encore totalement chargée et prête). Un sous-menu s'ouvrira, vous permettant de choisir entre différents types d'analyse de texte :

    • « Remue-méninges interactif avec l'IA » : une session de remue-méninges axée sur les questions concernant vos données.
    • « Reformuler et résumer » : une reformulation condensée, sans interprétation supplémentaire.
    • « Génération de thèmes » : extraction d'une liste de thèmes (basé sur Friese, S. (2024) : Prompting for Themes. https://community.qeludra.com/posts/prompts-for-qualitative-research-prompting-for-themes).
    • « Analyser l'inattendu » : recherche de nouvelles informations surprenantes et inattendues dans vos données.
    • « RSR reconstructive » : analyse complète basée sur la méthode documentaire (Source : Lieder, F. R. & Schäffer, B. (2024). Reconstructive Social Research Prompting (RSRP). Distributed Interpretation between AI and Researchers in Qualitative Research. https://doi.org/10.31235/osf.io/d6e9m).

    Note : Il n'est pas conseillé de sélectionner un document entier ou une très longue section de texte pour l'analyse. Cela conduira à des résultats superficiels et peut devenir coûteux avec le temps, car OpenAI vous facture en fonction de la quantité de texte envoyée avec chaque requête.

    Si vous souhaitez définir vos propres types d'analyse de texte, vous pouvez le faire en les ajoutant à la Bibliothèque de prompts IA (option avancée).

    Chat d'analyse de code

    • Analyse toutes les données collectées sous un code ou une catégorie particulière.
    • Utilisez-le dans les phases ultérieures de votre recherche pour discuter et consolider vos résultats.
    • Similaire à « l'analyse thématique » décrite ci-dessus, mais fonctionne sur vos données codées, et non sur une recherche libre de thème.

    Depuis l'onglet Chat IA, cliquez sur « Nouveau > Nouveau chat d'analyse de code ». Dans la fenêtre suivante, sélectionnez un code ou une catégorie que vous souhaitez analyser, et choisissez parmi plusieurs prompts analytiques :

    • « Résumé du code » vous donnera un aperçu des données associées au code sélectionné.
    • « Analyser l'inattendu » recherchera les informations les plus intéressantes et inattendues dans vos données codées.
    • « Analyser les différences » examinera s'il existe des différences significatives entre vos cas/documents empiriques concernant le code sélectionné.

    Notez qu'une analyse de code enverra toutes les données collectées sous ce code ou cette catégorie à l'IA, jusqu'à la longueur maximale d'entrée définie pour le modèle d'IA, ce qui représente environ 100 pages de texte dans le cas de GPT-4 turbo. Au moment de la rédaction (novembre 2024), l'envoi d'un texte de 100 pages à GPT-4 turbo pour analyse vous coûterait environ 60 cents US. Vos requêtes réelles seront probablement beaucoup plus petites et moins chères. Cependant, gardez à l'esprit le coût lors de l'analyse approfondie de codes avec l'IA.

    Chat général

    • Posez à l'IA n'importe quelle question non directement liée à vos données — en somme, un ChatGPT intégré.

    Utilisez-le pour des tests ou pour des séances de remue-méninges avec l'IA sur des théories, des méthodologies, ou même pour décider du prochain dîner avec l'équipe de recherche. Notez cependant que ce chat n'a pas accès à vos données et ne s'appuie que sur les connaissances internes du modèle de langage.
