    Instrucciones de Instalación y ejecución de QualCoder

    INSTALACIÓN

    Para las instalaciones manuales (mediante la línea de comandos o terminal), se hace referencia a una versión específica de lanzamiento (por ejemplo, la versión 3.8). Te recomendamos consultar la página de lanzamientos (releases) para obtener la versión más reciente, ya que podría existir una actualización posterior.

    Requisitos Previos

    VLC es opcional; no obstante, su instalación es importante si requieres realizar codificación de archivos de audio o video. En ciertos sistemas operativos (por ejemplo, Fedora), es posible que QualCoder presente dificultades para localizar la ruta de instalación de VLC. De manera opcional, se recomienda instalar FFmpeg para habilitar la visualización de formas de onda (waveform); para mayor información sobre este procedimiento, consulta: https://phoenixnap.com/kb/ffmpeg-windows.

    Windows

    Consulta: https://github.com/ccbogel/QualCoder/releases/3.8

    Para los archivos ejecutables generados en Windows 11, se dispone de las siguientes dos opciones:

    Opción 1: En la página de lanzamientos (releases) se encuentra disponible un instalador para Windows denominado: QualCoder_3_8_Win_Setup.exe (Ver imagen 1), descárgalo y ejecútalo. En caso de que se muestre la advertencia de Microsoft Defender SmartScreen, sigue los pasos de la imagen 3.

    Opción 2: Para utilizar la versión portable, descarga el archivo QualCoder_3_8_Win_Portable.exe y haz doble clic para abrirlo (Ver imagen 2). El inicio puede demorar unos 20 segundos. En caso de que se muestre la advertencia de Microsoft Defender SmartScreen, continúa con los pasos 2B y 3B de la misma imagen.

    Versión portable (Imagen 2):

    Al ejecutar el archivo .exe por primera vez (Ver imagen 3), el sistema operativo Windows solicitará tu autorización para iniciar QualCoder. Esta advertencia (Microsoft Defender SmartScreen) se genera debido a que el software proviene de un desarrollador no registrado. La obtención de un certificado de "editor de confianza" conlleva costos elevados, por lo cual no se prevé su adquisición en el futuro cercano.

    MacOS

    Instalación desde paquetes de aplicación (App bundles):

    Asegúrate de haber instalado todas las actualizaciones disponibles de macOS antes de iniciar el proceso de instalación. Dirígete a la aplicación Ajustes del Sistema en tu Mac. En la barra lateral, haz clic en General (es posible que debas desplazarte hacia abajo) y, posteriormente, selecciona Actualización de software.

    Instalación de QualCoder: Haz doble clic en el archivo .dmg descargado. Posteriormente, arrastra el icono de QualCoder hacia el acceso directo de la carpeta de Aplicaciones.

    Inicia QualCoder desde el Launchpad o haciendo doble clic en la aplicación dentro de tu carpeta de Aplicaciones. Actualmente, no disponemos de la capacidad para firmar los paquetes de aplicación (app bundles). Por lo tanto, al intentar ejecutar el programa por primera vez, es posible que el sistema muestre una advertencia emergente indicando que QualCoder proviene de un desarrollador no registrado, o bien el mensaje: "QualCoder no se puede abrir porque Apple no puede realizar un análisis en busca de software malicioso". Dependiendo del tipo de procesador de tu equipo, podrías visualizar una de las siguientes opciones:

    Si visualizas las opciones "Mostrar en el Finder" o "Aceptar", haz clic en Aceptar.

    Si visualizas las opciones "Listo" o "Trasladar a la papelera", haz clic en Listo.

    Posteriormente, dirígete a Ajustes y selecciona la sección Privacidad y seguridad. Desplázate hacia abajo hasta localizar un mensaje indicando que se impidió la ejecución de QualCoder. Haz clic en el botón "Abrir de todos modos". Es posible que el sistema te solicite ingresar tu contraseña y que debas confirmar la acción haciendo clic nuevamente en "Abrir". Tras validar tu identidad, espera a que el programa se inicie; este proceso puede demorar varios segundos.

    A partir de este momento, QualCoder debería ejecutarse sin inconvenientes.

    Instalación desde el código fuente (source code):

    Descarga el código fuente. Lee y aplica las instrucciones correspondientes al uso del código fuente en macOS, detalladas en el archivo Readme.md. Omite las comillas invertidas ( ` ) que delimitan las instrucciones de la línea de comandos, dado que estas se emplean únicamente con fines de formato y visualización.

    Descarga el código fuente. Lee y aplica las instrucciones correspondientes al uso del código fuente en Ubuntu, detalladas en el archivo Readme.md. Asegúrate de omitir las comillas invertidas ( ` ) que delimitan las instrucciones de la línea de comandos, dado que estas se utilizan exclusivamente con fines de formato y visualización.

    Fedora Linux

    Estas instrucciones descargan el código fuente actual directamente desde GitHub. Nota: La distribución Fedora utiliza Wayland, el cual podría presentar problemas de compatibilidad con la interfaz gráfica de Qt. Se recomienda instalar adicionalmente Xwayland para asegurar un funcionamiento óptimo.

    Descarga el código fuente. Lee y aplica las instrucciones correspondientes al uso del código fuente en Fedora, detalladas en el archivo Readme.md. Asegúrate de omitir las comillas invertidas ( ` ) que delimitan las instrucciones de la línea de comandos, dado que estas se utilizan exclusivamente con fines de formato y visualización.

    Es posible que se presenten problemas de compatibilidad en QualCoder al trabajar con archivos de audio o video en la distribución Fedora.
